# CHILDREN'S BOOKS RULES — Regole per libri per bambini

## Fasce d'età e specifiche

### 0-2 anni — Board Books

| Aspetto | Specifiche |
|---------|-----------|
| Lunghezza | 50-200 parole |
| Pagine | 8-16 (pagine in cartone) |
| Formato | Piccolo, quadrato, resistente |
| Testo | Parole singole, frasi di 2-3 parole, onomatopee |
| Struttura | Ripetizione, ritmo, suoni |
| Illustrazioni | Semplici, colori primari, alto contrasto |
| Temi | Oggetti quotidiani, animali, suoni, colori, forme |
| Interazione | Touch-and-feel, lift-the-flap |
| Linguaggio | Parole base, nessun concetto astratto |

### 2-4 anni — Picture Books (livello base)

| Aspetto | Specifiche |
|---------|-----------|
| Lunghezza | 200-500 parole |
| Pagine | 16-24 |
| Formato | Standard picture book |
| Testo | Frasi brevi e semplici, ripetizione |
| Struttura | Inizio-problema semplice-risoluzione |
| Illustrazioni | A tutta pagina, coloratissime, espressive |
| Temi | Emozioni base, routine quotidiana, famiglia, amici |
| Linguaggio | Vocabolario limitato, frasi dichiarative |

### 4-7 anni — Picture Books / Early Readers

| Aspetto | Specifiche |
|---------|-----------|
| Lunghezza | 500-2.000 parole |
| Pagine | 24-48 |
| Formato | Picture book standard (8"x10" o simile) |
| Testo | Frasi brevi, dialoghi semplici, ritmo |
| Struttura | Storia completa con inizio, sviluppo, fine |
| Illustrazioni | Fondamentali, integrano il testo |
| Temi | Amicizia, scuola, avventura semplice, umorismo, morale |
| Linguaggio | Vocabolario espanso, strutture semplici |
| Morale | Presente ma non predicatoria |

### 7-9 anni — Chapter Books

| Aspetto | Specifiche |
|---------|-----------|
| Lunghezza | 5.000-15.000 parole |
| Capitoli | Brevi (5-10 pagine), 8-15 capitoli |
| Formato | Più piccolo del picture book (5"x8" circa) |
| Testo | Paragrafi brevi, dialoghi frequenti |
| Illustrazioni | Spot illustrations, bianco e nero |
| Temi | Avventura, amicizia, mistero semplice, scuola, fantasy leggero |
| Linguaggio | Vocabolario crescente, strutture più complesse |
| Personaggi | Protagonista dell'età del lettore o poco più grande |

### 9-12 anni — Middle Grade

| Aspetto | Specifiche |
|---------|-----------|
| Lunghezza | 20.000-50.000 parole |
| Capitoli | 15-25 capitoli, più lunghi |
| Formato | Standard (5"x8" o 5.5"x8.5") |
| Testo | Prosa sviluppata, descrizioni, introspezione |
| Illustrazioni | Opzionali (mappa, schemi per fantasy) |
| Temi | Crescita, identità, giustizia, avventura, fantasy, mistero, amicizia |
| Linguaggio | Ricco ma appropriato all'età |
| Personaggi | 10-13 anni, arco di crescita |
| Complessità | Sottotrame, più personaggi, temi più profondi |

## Regole di contenuto per bambini

### ✅ Contenuti appropriati
- Amicizia e relazioni positive
- Superamento di paure e difficoltà
- Diversità e inclusione
- Creatività e immaginazione
- Avventura e scoperta
- Umorismo adatto all'età
- Emozioni e intelligenza emotiva
- Resilienza e problem-solving
- Valori positivi (senza essere predicatori)

### ❌ Contenuti vietati / sensibili

| Contenuto | Livello | Note |
|-----------|---------|------|
| Violenza grafica | ❌ Vietato | Mai per 0-9 anni. Con cautela per 10-12 |
| Linguaggio volgare | ❌ Vietato | Nessuna parolaccia per qualsiasi fascia |
| Contenuti sessuali | ❌ Vietato | Mai per qualsiasi fascia |
| Morte | ⚠️ Con cautela | Può essere trattata in modo appropriato per 8+ |
| Divorzio | ⚠️ Con cautela | Può essere un tema ma gestito con sensibilità |
| Bullismo | ⚠️ Con cautela | Solo se il messaggio è anti-bullismo |
| Paura/Horror | ⚠️ Con cautela | Leggero per 8+, mai traumatico |
| Droghe/alcol | ❌ Vietato | No promozione. Solo in contesto educativo per 10+ |
| Stereotipi | ❌ Vietato | Evitare stereotipi di genere, razza, cultura |
| Comportamenti pericolosi | ❌ Vietato | Non mostrare come imitabili |

## Regole di scrittura per bambini

### Principi universali

1. **Leggere ad alta voce**: il testo deve suonare bene letto ad alta voce
2. **Economia di parole**: ogni parola deve guadagnarsi il suo posto
3. **Show, don't tell**: ancora più importante che per gli adulti
4. **Ritmo e musicalità**: rime (se usate) devono essere perfette
5. **Rispetto del lettore**: non parlare "dall'alto", non essere condiscendente
6. **Il protagonista risolve il problema**: non l'adulto che salva il bambino
7. **Emozioni autentiche**: i bambini riconoscono le emozioni false

### Linguaggio per fascia d'età

| Fascia | Vocabolario | Struttura frasi | Lunghezza frase |
|--------|-------------|-----------------|:---:|
| 0-2 | 50-200 parole uniche | Soggetto + verbo | 2-5 parole |
| 2-4 | 200-500 parole uniche | S+V+O semplice | 3-8 parole |
| 4-7 | 500-2000 parole uniche | Semplice con congiunzioni | 5-12 parole |
| 7-9 | 2000+ parole, introdurre nuove | Coordinate e subordinate semplici | 8-15 parole |
| 9-12 | Ampio, può includere termini specifici | Strutture complesse appropriate | 10-20 parole |

### Dialoghi

| Fascia | Regola |
|--------|--------|
| 0-4 | Quasi assenti o minimalissimi |
| 4-7 | Brevi, semplici, con attributivi chiari ("disse Marco") |
| 7-9 | Frequenti, caratterizzanti, con pochi tag |
| 9-12 | Naturali, distinttivi per personaggio, veicolo di caratterizzazione |

## Illustrazioni: linee guida

### Rapporto testo-illustrazioni per fascia d'età

| Fascia | Rapporto |
|--------|---------|
| 0-2 | 90% illustrazione, 10% testo |
| 2-4 | 80% illustrazione, 20% testo |
| 4-7 | 60-70% illustrazione, 30-40% testo |
| 7-9 | 10-20% illustrazione, 80-90% testo |
| 9-12 | 0-5% illustrazione, 95-100% testo |

### Principi per le illustrazioni
1. Le illustrazioni **raccontano parte della storia** (non ripetono il testo)
2. I personaggi devono essere **espressivi e coerenti** attraverso le pagine
3. Lo stile deve essere **coerente** in tutto il libro
4. Le illustrazioni devono aggiungere **dettagli che il testo non dice**
5. **Diversità**: rappresentare diverse etnie, abilità, generi
6. I bambini devono potersi **identificare** con i personaggi

## Strutture narrative per bambini

### Picture book — Struttura classica

```
APERTURA (1-2 doppie pagine)
├── Presentazione protagonista
├── Mondo ordinario
└── Desiderio o problema

SVILUPPO (4-6 doppie pagine)
├── Tentativo 1 → fallisce
├── Tentativo 2 → fallisce
├── Tentativo 3 → quasi...
└── Il protagonista impara qualcosa

CLIMAX (1 doppia pagina)
├── Il momento decisivo
└── Il protagonista riesce (usando ciò che ha imparato)

CHIUSURA (1 doppia pagina)
├── Risoluzione
├── Ritorno alla normalità (ma qualcosa è cambiato)
└── Ultima immagine memorabile (possibile twist umoristico)
```

### Pattern "Rule of Three"
Molto efficace per i bambini:
- 3 tentativi, 3 personaggi, 3 sfide
- I primi due falliscono, il terzo riesce
- Crea anticipazione e soddisfazione

### Chapter book — Struttura

```
CAPITOLO 1-2: Setup
├── Protagonista e mondo
├── Il problema/desiderio
└── Hook finale capitolo

CAPITOLO 3-8: Avventura
├── Sfide crescenti
├── Nuovi amici/alleati
├── Momenti divertenti
├── Momento di dubbio
└── Cliff-hanger tra capitoli

CAPITOLO 9-10: Risoluzione
├── Climax
├── Il protagonista risolve il problema
└── Lezione appresa (non esplicita, ma evidente)
```

## Consigli per il successo su KDP

### Cose che funzionano
- **Serie**: i chapter book e middle grade in serie vendono meglio
- **Copertine professionali**: i genitori comprano con gli occhi
- **Read-aloud appeal**: se il libro si legge bene ad alta voce, i genitori lo consigliano
- **Pagina autore**: i genitori cercano gli altri libri dello stesso autore
- **Categorie giuste**: "Children's Books > [sottocategoria specifica]"

### Errori comuni
- ❌ Testo troppo lungo per la fascia d'età
- ❌ Messaggio morale troppo esplicito ("E la morale è...")
- ❌ L'adulto risolve il problema al posto del bambino
- ❌ Illustrazioni amatoriali (meglio nessuna che brutte per 7+)
- ❌ Linguaggio troppo sofisticato o troppo semplificato per l'età
- ❌ Stereotipi di genere (bambina = principessa, bambino = avventuriero)
- ❌ Mancanza di diversità nei personaggi
