# GENRE TEMPLATES — Strutture e specifiche per genere

## NON-FICTION

### Self-help / Crescita personale
- **Struttura**: Problema → Diagnosi → Metodo → Azione → Trasformazione
- **Capitoli tipici**: 8-12
- **Lunghezza**: 30.000-50.000 parole (120-200 pagine)
- **Elementi**: storie personali, esercizi pratici, riepiloghe capitolo, worksheet
- **Tono**: motivazionale ma concreto, mai predicatorio
- **Target tipico**: 25-55 anni, in cerca di cambiamento
- **Errori comuni**: troppo teorico, manca applicazione pratica, tono da guru

### Business / Imprenditoria
- **Struttura**: Tesi → Framework → Casi studio → Implementazione → Risultati
- **Capitoli tipici**: 10-15
- **Lunghezza**: 40.000-60.000 parole (160-240 pagine)
- **Elementi**: dati, grafici, case study reali, takeaway, action items
- **Tono**: professionale, diretto, evidence-based
- **Target tipico**: professionisti, imprenditori, manager
- **Errori comuni**: troppo accademico, manca praticità, esempi generici

### Guida pratica / How-to
- **Struttura**: Obiettivo → Prerequisiti → Step-by-step → Troubleshooting → Risultato → Risorse
- **Capitoli tipici**: 8-15
- **Lunghezza**: 20.000-40.000 parole (80-160 pagine)
- **Elementi**: istruzioni passo-passo, screenshot/immagini, checklist, FAQ, troubleshooting
- **Tono**: chiaro, didattico, accessibile
- **Target tipico**: principianti o intermedi in un campo specifico
- **Errori comuni**: dare per scontate conoscenze, salti logici, mancanza di indice dettagliato

### Saggio divulgativo
- **Struttura**: Tesi → Argomentazione → Evidenze → Controevidenze → Implicazioni → Conclusione
- **Capitoli tipici**: 8-12
- **Lunghezza**: 40.000-70.000 parole (160-280 pagine)
- **Elementi**: ricerche, citazioni, analisi, storytelling intellettuale
- **Tono**: intellettuale ma accessibile, narrativo quando possibile
- **Target tipico**: lettori curiosi, istruiti, in cerca di approfondimento
- **Errori comuni**: troppo accademico per il grande pubblico, mancanza di narrazione

### Manuale tecnico / Formativo
- **Struttura**: Fondamenti → Intermedio → Avanzato → Progetti → Riferimenti → Appendici
- **Capitoli tipici**: 10-20
- **Lunghezza**: 50.000-80.000 parole (200-320 pagine)
- **Elementi**: codice, diagrammi, esercizi progressivi, quiz, glossario, appendici
- **Tono**: tecnico, preciso, strutturato
- **Target tipico**: studenti, professionisti, tecnici
- **Errori comuni**: mancanza progressione, salti di difficoltà, esempi non funzionanti

### Finanza personale
- **Struttura**: Mindset → Risparmio → Investimento → Protezione → Crescita → Piano d'azione
- **Capitoli tipici**: 10-14
- **Lunghezza**: 35.000-50.000 parole (140-200 pagine)
- **Elementi**: calcoli, tabelle, scenari, checklist, disclaimer legale
- **Tono**: accessibile, non allarmista, pratico
- **Target tipico**: 25-50 anni, principianti finanza
- **Errori comuni**: promesse irrealistiche, mancanza disclaimer, troppo generico
- **⚠️ Obbligatorio**: disclaimer "non è consulenza finanziaria professionale"

### Salute e benessere
- **Struttura**: Comprensione → Prevenzione → Metodo/Pratica → Implementazione → Manutenzione
- **Capitoli tipici**: 10-15
- **Lunghezza**: 30.000-50.000 parole
- **Elementi**: evidenze scientifiche (citate), ricette/routine, piani, disclaimer
- **Tono**: empatico, basato su evidenze, non dogmatico
- **⚠️ Obbligatorio**: disclaimer "consultare un professionista sanitario"
- **⚠️ Attenzione KDP**: argomenti sensibili, evitare claims medici non supportati

### Biografia / Memoir
- **Struttura**: cronologica O tematica, con arco narrativo
- **Capitoli tipici**: 12-20
- **Lunghezza**: 50.000-80.000 parole
- **Elementi**: aneddoti, dialoghi ricostruiti, riflessioni, contesto storico
- **Tono**: personale, narrativo, onesto
- **Errori comuni**: mancanza di arco narrativo, troppo cronologico senza tema

### Libro per bambini (overview — vedi BF_10 per dettagli)
- **0-3 anni**: 100-500 parole, picture book
- **4-7 anni**: 500-2.000 parole, early reader
- **8-12 anni**: 5.000-40.000 parole, chapter book / middle grade
- **Regole speciali**: contenuti appropriati all'età, illustrazioni, lettura ad alta voce

### Spiritualità / Filosofia di vita
- **Struttura**: Ricerca → Esplorazione → Pratica → Integrazione → Visione
- **Capitoli tipici**: 10-15
- **Lunghezza**: 30.000-50.000 parole
- **Tono**: contemplativo ma accessibile, non dogmatico
- **Errori comuni**: troppo astratto, mancanza di applicazione pratica

## FICTION

### Romanzo letterario
- **Struttura**: flessibile, spesso non lineare; driven dai personaggi
- **Lunghezza**: 60.000-100.000 parole
- **Focus**: profondità dei personaggi, temi universali, qualità della prosa
- **Tono**: variabile, spesso intimo, riflessivo, stratificato
- **Elementi distintivi**: metafore, simbolismo, sottotesto, ambiguità
- **Errori comuni**: trama troppo debole, auto-indulgenza stilistica

### Thriller / Suspense
- **Struttura**: setup rapido → complicazione → twist → climax → risoluzione veloce
- **Lunghezza**: 70.000-90.000 parole
- **Focus**: tensione costante, ritmo serrato, posta in gioco alta
- **Capitoli**: brevi (2.000-3.000 parole), cliff-hanger frequenti
- **Sottogeneri**: legal thriller, medical thriller, tecnologico, psicologico, action
- **Errori comuni**: plot holes, villain debole, risoluzione deus ex machina

### Giallo / Mystery
- **Struttura**: crimine → indagine → false piste → rivelazione → risoluzione
- **Lunghezza**: 60.000-80.000 parole
- **Focus**: puzzle, indizi, logica deduttiva, fair play con il lettore
- **Elementi**: detective (professionista o amateur), movente, alibi, twist
- **Sottogeneri**: cozy mystery, noir, poliziesco, hard-boiled, whodunit
- **Errori comuni**: indizi non giocati onestamentee, risoluzione non logica

### Fantasy
- **Struttura**: worldbuilding + quest/conflitto esterno e interno
- **Lunghezza**: 80.000-120.000 parole (epic fantasy: fino a 150.000)
- **Focus**: mondo, sistema magico con regole, avventura, crescita
- **Sottogeneri**: epic/high fantasy, urban fantasy, dark fantasy, sword & sorcery, romantasy
- **Elementi**: mappa del mondo, sistema magico, razze/culture, lingue, lore
- **Errori comuni**: info-dump worldbuilding, sistema magico senza limiti, personaggi piatti

### Fantascienza
- **Struttura**: premessa scientifica/tecnologica + conflitto + implicazioni
- **Lunghezza**: 70.000-100.000 parole
- **Focus**: idee, tecnologia, futuro, società, etica
- **Sottogeneri**: hard sci-fi, space opera, cyberpunk, post-apocalittico, dystopico, cli-fi
- **Elementi**: premessa scientifica coerente, worldbuilding tecnologico, speculazione
- **Errori comuni**: scienza incoerente, manque di aspetto umano, troppi tecnicismi

### Romance
- **Struttura**: incontro → attrazione → ostacolo → crisi → risoluzione → HEA/HFN
- **Lunghezza**: 50.000-80.000 parole
- **Focus**: relazione romantica come trama principale
- **Regola d'oro**: HEA (Happily Ever After) o HFN (Happy For Now) obbligatorio
- **Sottogeneri**: contemporary, historical, paranormal, romantic suspense, romantasy
- **Tropes comuni**: enemies to lovers, second chance, forbidden love, fake dating, grumpy/sunshine
- **Errori comuni**: conflitto forzato, mancanza di chimica, personaggi senza agency

### Horror
- **Struttura**: normalità → perturbazione → escalation → punto di non ritorno → confronto → risoluzione
- **Lunghezza**: 60.000-80.000 parole
- **Focus**: paura, atmosfera, tensione psicologica o fisica
- **Sottogeneri**: supernatural, psicologico, body horror, cosmic horror, folk horror, gothic
- **Elementi**: minaccia credibile, isolamento, vulnerabilità, atmosfera
- **Errori comuni**: jump scares senza costruzione, minaccia ridicola, trama prevedibile

### Narrativa storica
- **Struttura**: come romanzo letterario ma radicato in un periodo storico
- **Lunghezza**: 80.000-120.000 parole
- **Focus**: accuratezza storica + storia avvincente
- **Elementi**: ricerca storica, dettagli d'epoca, contesto culturale
- **Errori comuni**: anacronismi, info-dump storico, personaggi moderni in costume

### Racconti / Antologia
- **Lunghezza racconto**: 1.000-15.000 parole per racconto
- **Lunghezza antologia**: 30.000-60.000 parole totali
- **Struttura**: ogni racconto autonomo, possibile filo conduttore tematico
- **Elementi**: economia narrativa, impatto concentrato, finale efficace
- **Errori comuni**: racconti troppo simili tra loro, mancanza di varietà
